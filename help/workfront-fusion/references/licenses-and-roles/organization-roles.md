# 組織とチームの役割

この記事では、Workfront Fusionの様々なチームおよび組織の役割で使用できるアクセス権と機能について説明します。

>[!NOTE]
>
>Workfront Fusionでは、組織の役割はチームの役割とは異なります。 組織とチームの役割について詳しくは、[組織とチームの概要](/help/workfront-fusion/set-up-and-manage-workfront-fusion/set-up-and-manage-orgs-and-teams/set-up-orgs-teams-and-users/org-and-team-overview.md)を参照してください。

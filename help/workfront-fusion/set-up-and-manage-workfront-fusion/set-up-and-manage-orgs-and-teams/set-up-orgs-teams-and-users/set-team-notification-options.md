# チーム通知オプションの設定

組織でAdobe統合シェルを使用している場合は、Adobe通知領域から通知を受け取ります。

組織がAdobe Unified Shellに移行されていない場合は、チームが受け取る通知を選択できます。 通知はチームレベルで設定されます。

## 通知オプションの設定

組織がAdobe Unified Shell上にない場合は、Fusionで直接通知を設定できます。

メール通知オプションは、チームレベルで設定されます。

1. 左側のナビゲーションパネルで、「**[!UICONTROL チーム]**」をクリック
1. 「**[!UICONTROL 通知オプション]**」タブを選択します。
1. チームが受け取る通知を有効にします。

   <table style="table-layout:auto"> 
    <col> 
    <col> 
    <tbody> 
     <tr> 
      <td role="rowheader">「[!UICONTROL Warning in scenario run]」</td> 
      <td> <p>シナリオの実行で警告が発生した場合にメールを受信</p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL Errors in scenario run]</td> 
      <td>シナリオ実行でエラーが発生した場合に、メールを受け取ります。</td> 
     </tr> 
     <tr> 
      <td role="rowheader"> <p>[!UICONTROL Scenario deactivation]</p> </td> 
      <td><p>シナリオが非アクティブ化された場合、メールを受信します。</p><p>場合によっては、パフォーマンスやその他の問題が発生しているために、Workfront Fusion エンジニアリングチームによってシナリオがディアクティベートされる場合があります。 この場合、Workfront Fusionで通知が届きません。 </p></td>

</tr>
</tbody>
</table>

通知オプションの変更は自動的に保存されます。
